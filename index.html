  • Fortune Teller

    A React.js App which uses conditional rendering to display a tarot card reading which can be selected by the user from the landing page.
    The tarot card data is passed to the components using props, and I utilised the useState hook so that the event listeners on the "Select Reading" buttons will set the state to the value specified within the onClick event, triggering the re-render of the app component, conditionally returning the appropriate component.
    This is a personal project of my own creation.

  • To Do List

    Created using EJS and MongoDB, this to-do list dynamically creates a new database collection when you add a new endpoint to the URL. I also used the Mongoose ORM to interact with my MongoDB.
    The title of the list is today's date by default, but changes to the list name when another endpoint besides the home route is accessed. When a checkbox is clicked, the item associated is deleted from the database. Created as part of my web development course with Udemy.

    To Do List The Codebase

  • Simon Game

    A classic memory game recreated using jQuery, specifically to be played in web browser.
    To commence play, press any key on your keyboard (an event listener awaiting a key press event starts the game). A coloured square will flash and the played has to remember the order that the squares flashed in. A counter keeps score of your level and displays it back to the player, and each button responds with a different sound and is animated when pressed.
